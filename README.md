# ☢️ 废土避难所 · Wasteland Shelter

一款废土生存/经营 H5 单机游戏。招募幸存者、探索网格化废土地图、建设避难所、研发科技、收集神器、派遣远征,在末日中重建文明。

**🎮 在线游玩**: https://zsrmoyanzsr.github.io/wasteland-shelter/

## 核心玩法循环
**探索地图 → 发现地标/POI → 派遣收集 → 建设升级 → 研发科技 → 更强地探索**,不断滚雪球,最终达成4种结局之一。

---

## 📋 功能清单 (v4.2 完整版)

### 🏠 基地经营
- **14 种设施**: 农场/净水/工坊/发电机/废料场/医疗室/围墙 + 温室/瞭望塔/军械工坊 + **4种蓝图解锁建筑**(水力发电机/生物实验室/反物质反应堆/自动化农场)
- **设施升级**: 每种可升 5 级,产出倍增
- **基地升级**: 升级主基地 → 人口上限 +2/级、资源容量 +30/级
- **居民分配**: 把居民分配到设施工作,特长+技能影响产出
- **虚弱机制**: 居民受伤(health<60轻伤/<30重伤)时设施产能下降、派遣产出减少

### 🧑 幸存者系统
- **8 种职业**: 医生/工程师/侦察兵/士兵/农夫/商人/科学家/猎人
- **6 种技能**: farm/craft/medical/combat/scavenge/social,影响设施产出+派遣+事件
- **养成**: 派遣/探索获得经验,升级提升技能值
- **虚弱/恢复**: 重伤效率降低且不能派遣,医疗室可恢复
- **不死机制**: 休闲向,血量0=重伤静养恢复,不会死亡
- **解雇**: 可解雇不需要的幸存者(装备归还仓库,最后1人不可解雇)
- **立绘**: 12 张 AI 生成的二次元角色立绘(内联 base64,零依赖)

### 🗺️ 网格化多地图探索
- **8 张大地图**: 营地废墟/郊区小镇/废弃医院/辐射工厂/军事禁区/地下水道/秘密实验室/核战地堡
- **大网格**: 每张120-208格,探索空间充足
- **网格 + 迷雾**: 三态清晰区分(迷雾❓/已揭示虚线/已踏足✓)
- **探索事件**: 踏入未知格触发事件(~20%密度,大地图不烦),82个事件(好/坏/中立)
- **25 个特殊地标**: 踩到立即触发独有奖励:
  - 📐废弃图纸(解锁建筑蓝图) · ⛲资源矿脉 · 🏚️隐藏避难所(招募)
  - ⛩️神秘祭坛(全队技能+1) · 💎宝箱 · 🧬基因舱 · ☢️核弹坑(稀有材料+神器)
- **解锁链**: 各地图有递进解锁条件(发现POI/资源门槛/派遣次数/科技要求)
- **传送**: 基地"外出探索"→地图选择→传送

### 🚀 派遣系统
- 选已发现 POI + 组队(1-4人) + 倒计时
- **队伍属性影响产出**: 工程师多拿零件、医生多拿药品、士兵多拿废铁
- **远 POI 高回报**: 距离越远时间越长,但产出倍率越高
- **物品掉落**: 按 POI 类型随机掉落材料/消耗品/组件
- **神器掉落**: 危险越高概率越大(3%-15%),S级神器改变游戏
- 70% 概率触发途中事件(5类),战利品+经验

### ✨ 神器系统
- **18 种神器**: 4等级(C精良/B稀有/A史诗/S传说),每种**存档内唯一**
- **按属性评级装槽**: 综合属性≥25或等级≥5 → 2槽,否则1槽
- **S级神器**: 方舟之钥(隐藏结局)/凤凰之羽(不死)/虚空水晶/永恒之火
- 神器属性加成计入派遣产出

### 🔬 科技树
- **3 条线 × 5 级**: 防御工程/工业革命/生命科学
- 消耗资源+材料研发,提供**永久增益**
- 防御线(力场穹顶免袭击)/生产线(物质复制器)/生物线(方舟计划)
- 每条线满级**触发对应结局**

### 🛒 流浪商队交易
- **稀有到访**: 每 6-9 游戏天到访,停留 2-3 天
- **智能方案**: 用富余资源换稀缺物资(汇率合理无套利)
- 3 个方案,限时决定

### ⚔️ 威胁系统
- **4 种威胁**: 辐射风暴/瘟疫/大规模入侵/极寒寒潮
- **消耗道具应对**: 使用辐射清除剂/抗生素/兴奋剂/止血剂抵御
- **硬扛不致死**: 不应对会扣血(到1为止)+扣资源+心情下降,但绝不致死

### 🎒 物品与背包
- **17 种物品**: 材料5/消耗品4/组件4/特殊3
- **消耗品可使用**: 背包中点击消耗品→选幸存者→回血/治疗
- 获取途径: 派遣掉落 + 探索事件 + 商队 + 地标

### 🏆 结局系统
- **4 种结局**: 战争领主/重建文明/方舟计划/末日真相
- 科技满级或获得方舟之钥触发,全屏结局画面
- **世代传承**: 达成结局后永久+5%产出,鼓励二周目

### 📜 任务与成就
- **7 个引导任务** + **14 个累计成就**(含神器收藏家/全地图/全能所长)
- **重建进度%**: 综合进度的元进度条

### 💾 健壮存档系统
- **版本化迁移**(v6): 老存档自动升级,永不丢
- **滚动备份**: 主档损坏自动从备份恢复
- **离线挂机**: 关闭后资源按时间结算(上限8小时)
- **存档码**: 导出WS-码,换设备粘贴恢复(纯前端零后端)

## 技术栈
- **Vite + 原生 JS** (无框架),单 Canvas 即时模式 UI
- **零美术资源**: 程序化绘制 + Emoji + AI立绘(内联base64)
- 立绘: pollinations.ai (FLUX) AI 生成

## 运行
```bash
npm install
npm run dev        # 启动 http://localhost:5180
npm run build      # 生产构建到 dist/
```

## 项目结构
```
src/
  main.js                  # 入口: 游戏循环/屏幕路由/模态路由/测试钩子
  engine/
    state.js               # 中心状态 + DEFAULTS + getResCap (版本v6)
    economy.js             # 经济tick/虚弱debuff/离线结算/转生加成
    save.js                # 版本化存档(迁移v1→v6/默认合并/备份恢复)
    worldUpdate.js         # 主角移动/网格探索/事件触发/地标发现
    caravan.js             # 流浪商队(到访/方案/交易/限时)
    inventory.js           # 物品背包(增删查/消耗品使用)
    techEngine.js          # 科技研发(检查/扣除/结局触发)
    artifactEngine.js      # 神器(掉落/唯一性/槽位/穿戴)
    threatEngine.js        # 威胁(触发/道具应对/不致死惩罚)
    avatarLoader.js        # 立绘加载(内联base64+外部覆盖)
  content/
    regions.js             # 8张地图/7种POI/网格/解锁/传送/地标注入
    landmarks.js           # 8种特殊地标定义+每地图配置
    artifacts.js           # 18种神器(SABC级)
    tech.js                # 科技树(3线×5级+结局)
    items.js               # 17种物品+派遣掉落表
    threats.js             # 4种威胁定义
    endings.js             # 4种结局定义+触发检查
    facilities.js          # 14种设施(10基础+4蓝图解锁)
    survivors.js           # 8职业/8特长/名字池/升级
    exploreEvents.js       # 82个探索事件
    events.js              # 5个派遣事件
    tasks.js               # 7个任务
    achievements.js        # 14个成就+重建进度
    avatars_inlined.js     # 12张立绘base64
  screens/
    screenStart.js         # 开始屏+存档码导出导入+世代显示
    screenHud.js           # 资源条(含背包入口)+侧栏/底部导航
    screenBase.js          # 基地屏+建造+升级+研究中心入口
    screenMap.js           # 地图屏+网格+迷雾+地标渲染
    screenRoster.js        # 居民屏+详情+分配+神器+解雇
    screenDispatch.js      # 派遣屏+组队+结算+物品/神器掉落
    screenTasks.js         # 任务屏+成就+无线电招募
    screenExploreEvent.js  # 探索事件模态
    screenTrade.js         # 商队交易模态
    screenTech.js          # 科技树+背包+神器选择+威胁+消耗品使用 模态
    screenEnding.js        # 结局画面
  ui/
    ui.js input.js theme.js avatar.js guideBanner.js
```
