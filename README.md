# 废土避难所 · Wasteland Shelter

一款废土生存/经营 H5 单机游戏。招募幸存者、探索网格化废土地图、建设避难所、派遣远征,在末日中重建文明。

## 核心玩法循环
**招募 → 派遣 → 收获 → 建设 → 更强**,不断滚雪球。

## 功能清单 (v2.0 完整版)

### 🏠 基地经营
- **7 种设施**: 农场(食物)、净水(水)、工坊(零件)、发电机(电力)、废料场(废铁)、医疗室(药品)、围墙(防御)
- **设施升级**: 每种可升 5 级,产出倍增
- **基地升级**: 升级主基地 → 人口上限 +2/级、资源容量 +30/级 (打通滚雪球)
- **居民分配**: 把居民分配到设施工作,特长+技能影响产出

### 🧑 幸存者系统
- **6 种职业**: 医生/工程师/侦察兵/士兵/农夫/商人,各有特长
- **6 种技能**: farm/craft/medical/combat/scavenge/social,真正影响产出/治疗/事件
- **养成**: 派遣/探索获得经验,升级提升技能值
- **状态**: 生命/饱食/心情,重伤(health<30)效率减半且不能派遣
- **不死机制**: 休闲向,血量0=重伤静养恢复,不会死亡
- **立绘**: 12 张 AI 生成的二次元女角色立绘(内联 base64,零依赖)

### 🗺️ 网格化多地图探索
- **5 张地图**: 营地废墟、郊区小镇、废弃医院、辐射工厂、军事禁区
- **网格 + 迷雾**: 三态清晰区分(迷雾❓/已揭示虚线/已踏足✓)
- **探索事件**: 踏入未探索格触发 **28 种事件**(14好14坏),有真取舍
- **解锁规律**: 医院=发现小镇POI、工厂=parts≥30、军事=3次派遣+发现医院POI
- **传送**: 基地"外出探索"→地图选择→传送,各图进度独立

### 🚀 派遣系统
- 选已发现 POI + 组队(1-4人) + 倒计时(近点快速/远点高回报)
- **队伍属性影响产出**: 工程师多拿零件、医生多拿药品、士兵多拿废铁——组队有策略
- **远 POI 高风险高回报**: 距离越远时间越长(37s→217s),但产出倍率越高(1.0→1.9)
- 70% 概率触发途中事件(5类:物资/幸存者/丧尸/掠夺者/陷阱)
- 战利品回报 + 队伍经验

### 📜 任务与成就
- **7 个引导任务**: 逐步引导发展
- **10 个累计成就**: 基于统计的长远目标(招募5人/探索12地点/全地图解锁等)
- **重建进度%**: 综合基地/人口/设施/探索的元进度条

### 🛒 流浪商队交易
- **稀有到访**: 每 6-9 游戏天一支商队到访,停留约 2-3 天后离去
- **智能方案**: 自动用你富余的资源(食物/水/废铁)换稀缺物资(零件/药品/电力),解决后期爆仓
- **高价但值得**: 商队价格比自产贵 ~40%,但能即时获得稀缺物,关键时刻值得交易
- **3 个方案**: 每次带来 3 个不同交易,可全做或放弃,限时决定

### 💾 健壮存档系统 (v2)
- **版本化迁移**: 加新字段/内容老存档永不丢、永不崩
- **深度默认值合并**: 自动补全所有缺失嵌套字段
- **滚动备份**: 主档损坏自动从备份恢复
- **离线挂机**: 关闭后资源按时间结算(上限8小时)
- **存档码跨设备转移**: 开始屏导出存档码(WS-开头),换设备粘贴即可恢复进度(纯前端零后端)

## 技术栈
- **Vite + 原生 JS** (无框架),单 Canvas 程序化绘制
- **扁平极简风**: 圆角卡片/色块/Emoji 图标,零美术资源
- 立绘: pollinations.ai (FLUX) AI 生成,内联 base64

- npm install 安装 vite + playwright
- 浏览器: 现代浏览器(支持 ES Module + Canvas 2D)

## 🚀 让别人玩(部署)
游戏是纯静态文件,零后端。详见 **[DEPLOY.md](./DEPLOY.md)**:
- **最快(30秒)**: Netlify Drop 拖 `dist/` 文件夹 → 得到公开网址

- **临时**: `npm run dev` 同WiFi下别人输你的IP

## 项目结构
```
src/
  main.js              # 入口: 游戏循环/屏幕路由/测试钩子
  engine/
    state.js           # 中心状态 + DEFAULTS + getResCap + 版本号
    economy.js         # 资源产出/消耗/医疗治疗/技能加成
    save.js            # 版本化存档(迁移链/默认合并/备份/验证)
    worldUpdate.js     # 主角移动/网格探索/事件触发/传送
    caravan.js         # 流浪商队交易系统(到访/方案/交易)
    avatarLoader.js    # 立绘加载(内联base64)
  ui/
    ui.js              # 即时模式 UI 原语
    input.js           # 输入处理(坐标映射)
    theme.js           # 配色/字号
    avatar.js          # 程序化头像兜底
  screens/
    screenStart/Hud/Base/Map/Roster/Dispatch/Tasks/ExploreEvent/Trade.js
  content/
    survivors.js       # 职业/特长/技能/生成器/升级
    facilities.js      # 设施定义
    regions.js         # 多地图/网格/解锁/POI
    exploreEvents.js   # 28个探索事件
    events.js          # 派遣事件
    tasks.js           # 任务定义
    achievements.js    # 成就+重建进度
    avatars_inlined.js # 立绘base64(自动生成)
```
